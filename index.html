This is a single player version of the classic TV show.

To play the game you need to move horizontally across the board by answering questions correctly.

If you fail to answer a question correctly, that particular hex will block your path and an alternate route will be required.

You can answer the questions in any order.

The game will end when you have successfully joined the two ends (a win), or if there is no longer a viable path across the board (a loss).

Before you can play you need a set of questions. A set of questions can be added using either the "Load Question" button (replacing the existing question set) or by merging sets of questions. By default there is no initial set of questions loaded. A limited set of questions in a text file is included.

All buttons can be accessed by clicking on the hamburger menu button (top left). These will not be accessible once a game has begun and will only be available again once the game has ended.

The format for creating a set of questions is a list of CSV values in a text file. Each line should be made up of 'Answer,Question,Option1,Option2,Option3,Option4'. One of the options should match the answer.

Use the 'Examine' button to check your list of questions before using them in game. It will allow you to examine each question in detail and look for errors. It will not allow you to make or save changes.

If you are still unsure on how to play please follow the link:'Classic Blockbusters episode'.

Good Luck. And enjoy!
